Setup:
Create images from Dockerfile:
	sudo docker build -t <imageName:tag> <dirWithDockerFile> # The tag is optional
Run docker image in a container:
	sudo docker run -i -t <imageName:tag> /bin/bash

List stuff:
Images:
	sudo docker images
Containers:
	sudo docker ps -a   # -f status=exited

Cleanup:
Remove docker container:
	sudo docker rm <containerID> 
Remove docker image:
	sudo docker rmi <imageName:tag>
